The file used for validation is input_file_tuples.csv available in the main channel
list of files:
- output: synthetic population extracted on the global population
- validation_APE: Average Percentage Error to the global extraction
- validation_RMSE: Root Mean Square of Errors to the global extraction
- output_multiple: synthetic population extracted iterating over spatial units
- validate_multiple_APE: Average Percentage Error to the synthetic population extracted iterating over spatial units
- validate_multiple_RMSE: Root Mean Square of Errors to the synthetic population extracted iterating over spatial units
